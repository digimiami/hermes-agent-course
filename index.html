Frequently Asked Questions

Everything you need to know before enrolling.

Do I need coding experience? ▼
No. The course is designed for beginners. You'll learn everything step-by-step. Basic terminal familiarity helps but is not required — we cover the fundamentals.
Does it work on Windows? ▼
Yes. We include a dedicated Windows/WSL setup module and a troubleshooting guide. Hermes runs on Windows through WSL2 (Windows Subsystem for Linux), which is free and easy to install.
Do I need an OpenAI API key? ▼
No. Hermes works with 20+ providers. You can use OpenRouter (free tier available), Anthropic Claude, DeepSeek, Google Gemini, or even run local models. We show you all the options.
Can I use DeepSeek, Claude, or local models? ▼
Absolutely. The course covers setup for all major providers: OpenRouter, Anthropic, OpenAI, DeepSeek, Google Gemini, and local models via Ollama. You can switch between them anytime.
How much does hosting cost? ▼
A VPS costs around $5/month (DigitalOcean, Hetzner, Vultr). API usage is typically $0-5/month depending on how much you use it. Total: roughly $5-10/month.
Can I use Hermes for trading, marketing, real estate, or business automation? ▼
Yes. The course includes advanced use cases for trading bots, content automation, customer support, research assistants, and more. You can adapt Hermes to almost any industry.
Is this beginner friendly? ▼
Yes. The first module starts with "what is an AI agent" and builds up from there. Every command is explained. If you can follow a recipe, you can complete this course.
What happens after I pay? ▼
You get immediate lifetime access to all 10 modules, bonuses, and future updates. You'll receive a welcome email with login details and a recommended learning path.
Is there a money-back guarantee? ▼
Yes. 30-day full money-back guarantee. No questions asked. If the course isn't right for you, email us and you'll get a full refund.

Keep API Keys Private

Store secrets in .env files, never commit them to Git. We show you the exact setup.

Use Environment Variables

Trading API Safety
